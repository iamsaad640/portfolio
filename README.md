# saad.run

Personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

**[Live Site](https://saad.run)**

## About

Software Engineer specializing in full-stack development and AI-powered systems. I architect production-grade platforms — from multi-tenant SaaS dashboards to RAG-based conversational agents.

## Tech Stack

- **Framework** — Next.js 16, React 19, TypeScript
- **Styling** — Tailwind CSS, Framer Motion
- **Backend** — Node.js, Python, FastAPI, NestJS
- **AI** — LangChain, LangGraph, OpenAI SDK, vLLM
- **Databases** — PostgreSQL, pgvector, MongoDB, Qdrant
- **Deployment** — Vercel

## Features

- Dynamic Island navigation (mobile)
- Orbiting skill visualization (desktop)
- Interactive project showcase with NDA support
- Working contact form with email delivery
- Fully responsive across all devices
- Smooth animations and transitions

## Getting Started

```bash
npm install
npm run dev
```

Create a `.env.local` file:

```env
SMTP_EMAIL=your-email@gmail.com
SMTP_PASSWORD=your-app-password
```
